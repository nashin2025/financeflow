# Plan: Add Help Center and Support Pages

## Overview
Add a comprehensive Help Center page and improve support-related functionality for the FinanceFlow application. The app already has Privacy Policy and Terms of Service pages, but needs a proper Help Center with FAQs, contact information, and app rating functionality.

## Current State Analysis
- **Existing Pages**: Privacy Policy (`/privacy-policy`), Terms of Service (`/terms-of-service`)
- **Current Links**: Settings page has duplicate entries linking to external URLs
- **Missing**: Dedicated Help Center page, proper contact support page, app rating page

## Proposed Implementation

### 1. Create Help Center Page (`/help`)
**Location**: `src/app/help/page.tsx`

**Features**:
- FAQ section with common financial app questions
- Getting started guide
- Troubleshooting section
- Contact support link
- App rating encouragement
- Links to Privacy Policy and Terms of Service

**Content Structure**:
- Hero section with search
- FAQ accordion component
- Quick action cards (Contact Support, Rate App, etc.)
- Footer links

### 2. Create Contact Support Page (`/contact`)
**Location**: `src/app/contact/page.tsx`

**Features**:
- Support ticket form
- Contact information (email, hours)
- FAQ shortcuts
- Live chat integration (placeholder)
- Support categories (Technical, Billing, Feature Request, etc.)

### 3. Create Rate App Page (`/rate`)
**Location**: `src/app/rate/page.tsx`

**Features**:
- App store links (iOS, Android, Web)
- User feedback form
- Rating system (stars)
- Review submission
- Social sharing options

### 4. Update Settings Page
**Location**: `src/app/settings/page.tsx`

**Changes**:
- Remove duplicate entries
- Update links to point to new internal pages instead of external URLs
- Organize support links in a cleaner section

### 5. Create Shared Components
**Location**: `src/components/help/`

**Components**:
- `FAQAccordion.tsx` - Reusable FAQ component
- `SupportCard.tsx` - Quick action cards
- `ContactForm.tsx` - Contact form component
- `RatingStars.tsx` - Star rating component

### 6. Update Navigation/Layout
**Consider**: Add Help Center link to main navigation or footer if it exists

## Implementation Decisions (Based on User Confirmation)

### Feature Defaults
1. **Search functionality**: Include a basic search bar for FAQs with keyword filtering
2. **Contact methods**: Email support with categorized contact forms (Technical, Billing, Feature Request, General)
3. **App rating**: Redirect to app stores with review collection capability
4. **FAQ content**: Cover core financial topics (budgeting, goals, transactions, accounts, analytics)
5. **Feedback system**: Include a simple feedback form on the contact page

### Technical Specifications
- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS with existing design system
- **Components**: Shadcn/ui components for consistency
- **Forms**: React Hook Form for validation
- **Icons**: Lucide React icons
- **Responsiveness**: Mobile-first design approach
